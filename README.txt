-----DESCRIPTION-----
This program simulates a randomly generated race between a tortoise and a hare. 
It displays a play-by-play for every time one of the animals passes the other.
Use this program to demonstrate how random operates, practice function calls/returns,
    work between source files, and communicate with a user with printf and scanf.

-----COMPILE INSTRUCTIONS-----
gcc -Wall race.c animals.c -o theRace

-----PROGRAM INSTRUCTIONS-----
./theRace
Input an integer
